Mi az a repository

A githubon vannak branchek. Azok egy bizonyos résznek a nevei. Ezekbe lehet fájlokat pusholni, pullozni. A brancheket lehet mergelni is ez azt jelenti össze lehet a tartalmukat fűzni.

# Név Command Leírás
1 Pull git pull -u origin master Ez a master brachben tartozó fájlokat másolja a local repositoryba
2 Commit git commit -m "---" Ezt a commandot használjuk a push előtt ezzel adjuk meg milyen változtatásokat csináltunk
3 Push git push -u origin master A master nevezetű brachba küldjük a frissitett fájlokat vagy éppen újjat adunk hozzá
4 Status git status Megmutatja milyen státuszban van a fájl
5 Add git add _ A _ helyére lehet írni a .-ot ami minden fájlt hozzá ad, de ha csak a fájl nevét írjuk oda az is jó
